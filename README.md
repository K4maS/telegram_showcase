## Запуск проекта
1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/K4maS/telegram_showcase.git
   ```
2. Установить зависимости:
    ```bash
    npm install
    ```
3. Запустить сборку проекта:
      ```bash
    npm run build
      ```
4. Для локального запуска использовать:
   ```bash
    npm start
   ```
